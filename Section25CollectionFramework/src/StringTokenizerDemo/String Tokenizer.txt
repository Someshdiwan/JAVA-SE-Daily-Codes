Key-Value Pair.

Each pair follows the format: key=value

Different pairs are separated using the ';' delimiter.

Example:

- "name=Vijay" ? key = "name", value = "Vijay"
- "address=delhi" ? key = "address", value = "delhi"
- "country=india" ? key = "country", value = "india"
- "dept=cse" ? key = "dept", value = "cse"

Delimiters:
- '=' separates the key and value.
- ';' separates different key-value pairs.

Purpose:
- If a string contains structured data separated by delimiters,
we can extract and process the data using string tokenization.

Name=Token and Vijay=Token....

= & ; it is called as Deli-meters.

If String is having some data and separated by the deli-meter
then we can take out the data in the form of token.
